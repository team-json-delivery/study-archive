# 머리로 일한 시간, 몸으로 일한 시간
- 아래와 같이 실험 -> 혼자는 소음에 대한 피해자, 나머지는 소음 생성
  - 혼자 일하는 시간: 30%
  - 2명이서 일하기: 50%
  - 3명이상 함께 일하기: 20%
- 혼자 일하는 사람들은 소음으로 인해 피해를 받음
  - 혼자 일하는 그시간이 실제로 일하는 시간
  - 나머지 시간은 부가적인 활동, 휴식, 수다
##### 몰입
- 공학, 설계, 개발 등과 관련된 직종은 몰입이 필수
  - 몰입했을 때만 진도가 잘 나감
- 몰입은 스위치처럼 껐다 켰다 할 수가 없음 -> 방해가 잦은 환경에서는 몰입이 힘듬
- 본격적으로 몰입에 빠져들기까지 15분정도 소요
  - 몰입으로 빠져드는 와중에는 실제로 일하지 않음
##### 몰입하지 못하는 상태의 지속
- 전화 시간 5분, 몰입 준비 시간 15분을 합치면 총 20분 -> 12통이면 반나절 날아감
- 일할 시간을 잃어버리면 좌절감에 시달림
- 몰입을 빼앗아버리는 어떤 방해불도 작업 효율성과 작업 만족감을 줄일 것
##### 몰입 시간 측정
- 개발 업무에 투입되는 시간을 현실적으로 파악하려면 몰입이라는 현상 이해 필요
  - 집중해서 일하는 시간이 더 중요 -> 몰입 시간 60분 > 6분씩 10번
- 몰입시간 기록 시스템은 방해받지 않은 업무시간을 기록
  - 정직한 자료를 얻으려면 너무 적게 몰입했다고 문책 X
  - 몰입을 많이 못했다면 몰입할 환경을 못준 회사 책임
- 몰입 시간을 기록하는 업무 시간 관리 시스템 2가지 장점
  - 직원들이 몰입 시간의 중요성에 주목
    - 방해받지 않은 시간에 대한 중요성 인식 -> 자발적으로 확보하려고 노력함
  - 업무에 투입된 실제 시간이 기록
    - 몰입 시간 3000시간 필요하면 현재까지 몰입 시간 2000시간이면 2/3 정도 완료
##### 환경 지수
- 환경 지수: 방해받지 않은 시간이 근무 시간에서 많은 비중 차지
  - 방해받지 않은 시간 / 출석 시간
- 한 회사 내에서도 환경지수가 다름
  - 큰 정부 기관에서 0.38 ~ 0.10까지 나옴
- 환경 지수는 현 실정에 위협적일 수 있음
  - 생산성 비교로 인해 멋진 '절약'을 몽땅 위험에 빠뜨리게 됨
##### 빨간 두건의 정원
- 보통 사람들은 분위기 상 주변 사람들에게 아주 짧은 시간 동안이라도, 대놓고 방해 받기 싫다고 말하기 어려움
  - 환경 지수를 조금만 강조해도 조직 문화가 바뀜
##### 회사에서 생각하기
- 오늘날 다수의 사무실은 소음과 방해가 너무 많아 신중한 생각이 사실상 불가능
  - `조그만 평화롭고 조용하면` 사람들은 자신들의 두뇌를 더 많이 사용함

# 전화
- 하루의 몇 통의 전화로 몰입에 걸리는 시간 때문에 하루가 훌쩍 가버림
##### 상상 속 세계
- 전화에 대해 너무나 방해가 되는 장치라고 판단 되는 이야기
  - 혹해서 설치했다가는 아무도 일하지 못하리라 판단
##### 크리프트 스토리
- 우리가 전화에 얼마나 많은 시간을 기꺼이 내주는지 깨닫는 단계가 중요
  - 대화 도중 전화가 오면 대화를 중단하고 전화를 먼저 받음 -> 무례함
  - 위와 같은 행동이 잘못되었다는 것을 깨닫지 못함
- 관리자는 최소한 전화라는 방해물이 일에 몰두하려는 직원에게 미치는 영향을 경계해야함
##### 수정된 전화 윤리
- 해결책은 방해와 전화를 대하는 새로운 태도
  - 몰입해 일하려면 울려대는 전화를 무시하는 효율적이고 용납가능한 방법 필요
    - 용납: 가끔 일부러 전화를 받지 않는 사실을 기업 문화가 인정
    - 효율: 작업을 계속 진행하기 위해 전화벨 소리가 끊길 때까지 기다릴 필요가 없어야 함
- 전화는 방해지만 이메일은 아니라는 점
##### 호환 불가능한 멀티태스킹
- 설계와 같이 고도의 집중이 필요한 일을 하는 사람에게 방해는 생산성 살인임
- 태도 변화는 어떤 책략보다 중요
  - 때로 전화를 안받아도 된다는 사실을 알아야함
  - 시간의 양이 아니라 질이 중요

# 사무실에 문을 달자
- 합리적인 사무실 환경을 만드는 성공의 지표 중 하나는 문
  - 문은 소음과 방해를 조절
  - 실패를 가장 잘 보여주는 지표는 사내 방송
- 해당 지표들을 개선하라 -> 사람들이 일하기 시작함
##### 뚱뚱한 여가수가 노래를 부를 때까지 공연은 끝나지 않음
- 사무실의 몰락은 피해자들의 동의가 있었기에 가능 -> 침묵
- 이성적인 업무환경으로 가는 첫단계는 반복적인 주장 -> 사무실 환경이 생산성을 떨어뜨린다고 말하자
  - 사람들이 소음과 방해에 좀더 신중해지면서 환경이 좋아짐
  - 희생자들이 암묵적 동의를 철회함
- 반대의견도 제시될꺼임
  - 사람들은 화려한 사무실에 신경 쓰지 않음
  - 사무실을 뒤엎지 않고서도 해결할 더 저렴한 방법 존재
  - 문 닫힌 사무실은 활기찬 환경에 적합하지 않음
##### 사무실 치장 문제
- 사람들이 외관에 까다롭지 않다는 사실을 환경에 신경쓰지 않는다는 의미로 오해하기 쉬움
  - 소음, 사생활, 탁자, 책상 등
- 외관을 지나치게 강조하지만 핵심은 일하기 좋은 사무실인지 여부 -> 필수 요소
  - 생산성 손실이라는 대가를 치룸
##### 창의적인 공간
- 무자크나 다른 형태의 분홍색 잡음 발생기 설치 -> 적은 비용으로 소음 상쇄
  - 사람들이 각자의 아이포드나 헤드폰으로 소음을 차단함
  - 직원들의 창의력을 떨어뜨리는 효과 -> 생산성 대가를 치룸
- 지식 노동자의 대다수의 업무는 좌반구의 직렬처리를 거치지만, 모든 업무가 그렇지는 않음
  - 음악 듣는건 총체적인 우반구에서 소화 -> 창의적 도약은 우반구 맡음
- 환경으로 인한 저하되는 창의성은 은밀함 -> 오랜 기간 축적됨
  - 조직은 효율성이 떨어지고, 사람들은 마지못해 일하고, 우수한 인력들이 떠나감
##### 활기찬 사무실
- 닫힌 사무실이 1인용을 뜻하지는 않음 -> 2~3인용 사무실이 더 합리적임
- 함께 소통 모드가 있거나 동시에 몰입 모드에 빠져들므로 소음 충돌이 적어짐
##### 회사의 틀을 깨버리자
- 최고의 환경은 찍어내듯 복제할 수 없음
  - 한 사람에게 좋은 환경이 다른사람에게 좋지 않을수 있음
- 사람들이 필요로 하는 공간을 만들고 통일성은 사라짐 -> 원하는 대로 되면 사무실 문제를 잊고 일에 몰두

# 한 걸음씩 걷기
- 이상적인 환경 특정 살펴보기
  - 크리스토퍼 알렉산더라는 유명한 건축가를 통해 알아보자
##### 유기적 질서라는 알렉산더의 개념
- 사옥을 짓는데 첫번 째 단계는? 종합 계획 수립 -> 치명적인 탈선
  - 동일한 요소로 거대한 전체를 만드는 모듈 방식과 복제 -> 누구에게도 적합하지 않은 공간과 개성 부족한 통일성
  - 전체주의 질서를 세우려는 시도
- 알렉산더는 메타-계획을 제안 -> 시설 사용자 요구에 따라 진화해 나간다는 철학
  - 조금씩 커지는 성장 철학
  - 성장을 지배하는 패턴 또는 공유된 설계 원칙
  - 공간 사용자가 설계에 직접 관여
- 개인의 요구와 전체의 요구가 완벽하게 균형을 이룰때 자연적 질서 또는 유기적 질서가 나타남
##### 패턴(최악으로 나쁜 형태 네가지를 겨냥한 패턴 소개)
- 패턴 1: 조립 키트를 사용한 맞춤식 작업 공간
  - 모듈식 칸막이는 타협의 걸작
    - 고립된 느낌과 소음 방지도 안됨
    - 질 낮은 공가이면, 팀에게도 무용지물
  - 대안으로 팀을 둘러싸는 공간
    - 팀마다 공용 공간과 준 공용 공간 필요, 개인에게 사적 공간 필요
- 패턴 2: 창문
  - `창문 없는 방은 교도소와 같음`
  - 창문을 비용 때문에 많이 넣기가 힘들다? 평방미터당 공간 비용은 기존과 크게 다르지 않음
  - 비용은 눈에 잘 띄는 범주에 속하는 반면, 이익은 잘 측정되지 않아 거의 보이지 않은 범주에 속함
    - 비용: 공간과 서비스
    - 이익: 생산성 증가와 이직률 하락
- 패턴 3: 실내와 실외 공간
  - 실내 공간과 실외 공간을 통합하기도 쉬움
    - 햇살 좋은 날 어떤 사람은 테라스에서 일하고, 어떤 사람들은 정원이나 정자에서 일함
- 패턴 4: 공용 공간
  - 실내 들어섰을 때 점진적인 친밀도의 증가를 보이는 패턴
  - 사무실 입구에는 팀 전체가 사용하는 공간 필요
    - 일종의 난로 주변 역할을 하는 공간 -> 소통과 교류 공간
##### 패턴의 패턴
- 성공적인 공간에서 반복적으로 튀어나는 패턴은 인간 창조물의 특성과 본질적으로 일치하는 경향 존재
  - 사람으로서 할 일을 하게 만듬
  - 개인이자 그룹 일원이라는 사람의 존재감을 강조 -> 개인의 원래 모습 존중
##### 현실로 돌아오기
- 전직원까지 가지 않아도 팀원들의 문제만 해결해도 대단한 성과
  - 팀의 생산성이 높아지고 이직률이 낮아지면 관리자로서 능력 입중
- 프로젝트 팀이나 업무 팀을 회사 공간 밖으로 옮기면 거의 항상 결과가 좋음
